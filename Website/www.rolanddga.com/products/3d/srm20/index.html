Introduction to Roland MonoFab
monoFab Overview

Built on a rich heritage of Roland milling machines, the SRM-20 was designed from the ground up with a number of technological advancements that make it the easiest to use, most precise CNC desktop mill in its class. The drive system features stepper motors that micro-step for clean, precision contours. Engineered for optimum efficiency and productivity, this next-generation desktop mill boasts a feed rate that is two times faster than previous MDX-15/20 models. A new independent collet system allows for faster setting of the Z-axis base point. The VPanel computer interface provides control over X,Y, Z axes as well as feed rate and spindle speed. The fully enclosed cover prevents dust from escaping and minimizes noise level.
